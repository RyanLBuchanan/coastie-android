**Coastie** is a demonstration Android application for the **Coastal CTL AI Assistant**, designed to showcase ethical, student-first AI support for higher-education faculty.

This app mirrors the functionality and *design language* of the Coastie web experience, while using **native Android patterns (Jetpack Compose)** and an existing **Netlify + Azure AI Foundry backend**.

## üéØ Purpose

Coastie helps educators:

- Improve **Canvas course accessibility**
- Design **AI-resilient assessments**
- Create **rubrics and integrity notes**
- Model **ethical AI use** that supports learning (not graded work completion)

The Android app exists to:
- Demonstrate Coastie in **mobile and tablet contexts**
- Provide a **native UI** alternative to the web app
- Support **in-person demos** and pilot discussions
