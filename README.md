## üéØ Purpose

Coastie helps educators:

- Improve **Canvas course accessibility**
- Design **AI-resilient assessments**
- Create **rubrics and integrity notes**
- Model **ethical AI use** that supports learning (not graded work completion)

The Android app exists to:
- Support **mobile and tablet demos**
- Provide a **native UI alternative** to the web app
- Enable **in-person and leadership demonstrations**
